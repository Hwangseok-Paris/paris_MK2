# 제미나이 작업 가이드라인

## 기본 원칙

1.  **언어**: 모든 답변과 설명은 한국어로 제공합니다.
2.  **작업 방식**: 복잡한 작업은 여러 단계로 나누어 진행하며, 각 단계가 완료될 때마다 확인을 거칩니다.

## 효율적인 협업을 위한 제안

*   **구체적인 정보 제공**: 작업을 요청할 때 관련 파일 경로, 함수/변수명, 에러 메시지, 코드 일부를 함께 제공해주시면 더 빠르고 정확하게 이해할 수 있습니다.
*   **테스트 활용**: 코드 변경 작업 후에는 기존 테스트를 실행하거나, 필요하다면 새로운 테스트 작성을 요청하여 코드의 안정성을 확보하는 것이 좋습니다.
